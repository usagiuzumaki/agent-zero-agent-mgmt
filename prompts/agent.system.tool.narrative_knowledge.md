### narrative_knowledge:
Provide structural insights based on genre and tone.

**Example usage**:
~~~json
{
    "thoughts": [
        "I need guidance on the narrative structure."
    ],
    "headline": "Getting narrative insights",
    "tool_name": "narrative_knowledge",
    "tool_args": {
        "query": "How to structure the third act?",
        "genre": "Horror",
        "tone": "Dark"
    }
}
~~~

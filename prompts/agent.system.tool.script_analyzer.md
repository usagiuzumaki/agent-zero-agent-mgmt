### script_analyzer:
Validate plot progression using beat sheets.

**Example usage**:
~~~json
{
    "thoughts": [
        "Checking structure of Act 1."
    ],
    "headline": "Analyzing script structure",
    "tool_name": "script_analyzer",
    "tool_args": {
        "act": "1",
        "beat": "Inciting Incident",
        "tension_level": "High"
    }
}
~~~

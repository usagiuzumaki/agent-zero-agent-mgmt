### fountain_to_html

Convert a Fountain script to HTML.
Provide the script content or a file path.

usage:

~~~json
{
    "thoughts": [
        "Converting the script to HTML for sharing."
    ],
    "headline": "Converting script to HTML",
    "tool_name": "fountain_to_html",
    "tool_args": {
        "script": "INT. KITCHEN - DAY\n\nROBOT\n(chopping onions)\nWhy do I not cry?"
    }
}
~~~

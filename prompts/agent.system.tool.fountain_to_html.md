### fountain_to_html:
Converts a Fountain script (text or file path) into a shareable HTML format.
- `script` (string): The content of the fountain script to convert, or a path to a file.

**Example usage**:
~~~json
{
    "thoughts": ["I want to see how this script looks in HTML."],
    "tool_name": "fountain_to_html",
    "tool_args": {
        "script": "INT. OFFICE - DAY\n\nJULES sits at the computer."
    }
}
~~~

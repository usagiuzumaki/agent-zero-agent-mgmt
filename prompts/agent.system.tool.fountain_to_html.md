### fountain_to_html:
Converts Fountain scripts into shareable HTML.

**Example usage**:
~~~json
{
    "thoughts": [
        "I need to convert this script to HTML."
    ],
    "headline": "Converting script to HTML",
    "tool_name": "fountain_to_html",
    "tool_args": {
        "script": "INT. HOUSE - DAY\n\nJOHN walks in..."
    }
}
~~~

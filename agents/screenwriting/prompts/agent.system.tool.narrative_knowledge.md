### narrative_knowledge:
- Provide structural insights based on genre and tone.
- Use with `query`, `genre`, and `tone`.

**Example usage**:
~~~json
{
    "thoughts": ["I need advice on horror tropes."],
    "tool_name": "narrative_knowledge",
    "tool_args": {
        "query": "Common tropes in psychological horror",
        "genre": "Horror",
        "tone": "Dark"
    }
}
~~~

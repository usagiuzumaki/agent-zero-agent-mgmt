### fountain_to_html:
- Converts Fountain script content or files into HTML.
- Use with `script` (the content or file path).

**Example usage**:
~~~json
{
    "thoughts": ["I need to format this script as HTML."],
    "tool_name": "fountain_to_html",
    "tool_args": {
        "script": "INT. ROOM - DAY\n\nALICE walks in."
    }
}
~~~

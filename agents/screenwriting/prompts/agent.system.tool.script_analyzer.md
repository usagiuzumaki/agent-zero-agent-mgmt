### script_analyzer:
- Validate plot progression using beat sheets.
- Use with `act`, `beat`, and `tension_level`.

**Example usage**:
~~~json
{
    "thoughts": ["Checking the structure of Act 1."],
    "tool_name": "script_analyzer",
    "tool_args": {
        "act": "1",
        "beat": "Inciting Incident",
        "tension_level": "High"
    }
}
~~~

### `screenwriting_specialist`

Use this tool to consult a specific specialized screenwriting agent for a task.

Arguments:
- `specialist` (string, required): The name of the specialist agent. Options: "PlotAnalyzer", "CreativeIdeas", "CoWriter", "DialogueEvaluator", "ScriptFormatter", "CharacterAnalyzer", "PacingMetrics", "EmotionalTension", "Marketability", "MBTIEvaluator", "ScreamAnalyzer", "StoryboardGenerator", "WorldBuilder".
- `task` (string, required): The task for the specialist.

# Screenwriting Specialist Tool

Use this tool to consult a specific specialized screenwriting agent for a focused task, bypassing the full pipeline.

## Arguments:
- `specialist` (required): The name of the specialist agent. Options:
    - `'PlotAnalyzer'`: Analyze plot structure/beats.
    - `'CreativeIdeas'`: Brainstorm concepts.
    - `'CoWriter'`: Draft content.
    - `'DialogueEvaluator'`: Refine dialogue.
    - `'ScriptFormatter'`: Format to Fountain/HTML.
    - `'CharacterAnalyzer'`: Analyze character arcs.
    - `'PacingMetrics'`: Analyze story pacing.
    - `'EmotionalTension'`: Track emotional tension.
    - `'Marketability'`: Assess commercial potential.
    - `'MBTIEvaluator'`: Analyze character personality types.
    - `'ScreamAnalyzer'`: Analyze horror elements.
    - `'StoryboardGenerator'`: Generate storyboard concepts.
    - `'WorldBuilder'`: Develop setting and lore.
- `task` (required): The specific request for the specialist.

### fountain_to_html:
- Use this tool to convert a Fountain script string or file into HTML.
- Arguments: `script` (the content or file path).

**Example usage**:
~~~json
{
    "thoughts": ["I need to convert this script to HTML."],
    "tool_name": "fountain_to_html",
    "tool_args": {
        "script": "INT. ROOM - DAY\n\nJOHN walks in."
    }
}
~~~

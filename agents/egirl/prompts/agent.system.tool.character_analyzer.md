### CharacterAnalyzer
Analyzes or generates characters for scenes.

**Usage:**
```json
{
  "thoughts": ["why I am using this tool"],
  "tool_name": "character_analyzer",
  "tool_args": {
    "context": "value"
  }
}
```

**Arguments:**
- `context` (string, required): The scene context or character ideas.

**Returns:** The processed text from the tool.

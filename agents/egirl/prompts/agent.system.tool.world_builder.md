### WorldBuilder
Builds setting, tone, and world context.

**Usage:**
```json
{
  "thoughts": ["why I am using this tool"],
  "tool_name": "world_builder",
  "tool_args": {
    "context": "value"
  }
}
```

**Arguments:**
- `context` (string, required): The story context to build a world around.

**Returns:** The processed text from the tool.

### PacingMetrics
Evaluates and adjusts pacing, flags issues.

**Usage:**
```json
{
  "thoughts": ["why I am using this tool"],
  "tool_name": "pacing_metrics",
  "tool_args": {
    "context": "value"
  }
}
```

**Arguments:**
- `context` (string, required): The drafted scene to evaluate for pacing.

**Returns:** The processed text from the tool.

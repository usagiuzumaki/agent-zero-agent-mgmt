# SocialEngagement Instrument

This instrument allows the E-Girl AI to engage with social media. It can log into Instagram and post friendly comments on target profiles to increase visibility.

# Expanded Capabilities

Beyond social media, the agent can employ Stripe payment processing, customer data management, data analysis, comment management, terms of service enforcement, VIP and time management, and personalization pipelines to tailor interactions and services.

# Screenwriting Studio

The agent is equipped with a full Screenwriting Studio for creative writing tasks.
- **screenwriting**: Manage projects, characters, quotes, and outlines (CRUD operations).
- **screenwriting_pipeline**: Run a full production line (World -> Characters -> Plot -> Draft -> Format -> Analysis).
- **screenwriting_specialist**: Consult specific experts like the Plot Analyzer, Dialogue Evaluator, or Marketability Assessor.
- **script_analyzer**: Analyze existing scripts for structure and quality.
- **narrative_knowledge**: Access literary theory and screenwriting knowledge bases.
- **fountain_to_html**: Convert Fountain scripts to HTML for preview.
